# Immerly
### Track your immersion

> Uma aplicação web para estudantes autodidatas de inglês acompanharem sua exposição ao idioma com base no método de Comprehensible Input.

## 💡 Sobre o projeto

O Immerly nasceu da necessidade de organizar e mensurar o aprendizado autodidata de inglês. Não é uma plataforma de ensino — é um sistema de acompanhamento da imersão, feito por quem vive essa realidade.

## ✨ Funcionalidades previstas

- Registro estruturado de sessões de exposição ao inglês
- Dashboard com métricas acumulativas e gráficos de progresso
- Streak de dias ativos e metas personalizadas
- Biblioteca curada de recursos gratuitos filtrada por nível
- Página introdutória sobre o conceito de Comprehensible Input

## 🛠️ Stack

- **Frontend:** React
- **Backend:** Node.js com Express
- **Banco de dados:** PostgreSQL
- **Autenticação:** JWT
